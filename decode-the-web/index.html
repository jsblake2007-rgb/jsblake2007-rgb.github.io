02 — Choose Your Path

Three ways to build the web

Every site you've ever visited is the work of three overlapping crafts. Most developers start with one and grow into the others.

Front-End

Everything the user sees and touches. You sculpt layouts, motion, and interaction with HTML, CSS, and JavaScript — turning designs into living interfaces.

Back-End

03 — Under the Hood

The 100-millisecond journey of a click

Press Enter on a URL and a request sprints across the planet and back before you finish blinking. Here's the route every data packet takes.

  1. 1

    Your device

    Your browser breaks the request into data packets — small, addressed parcels of information — and hands them to your network.

  2. 2

    Router

    The router is your local traffic controller. It connects the computers on your network and makes sure each packet leaves through the right door.

  3. 3

    Modem & ISP

    Your modem bridges your LAN to the wider world. Your Internet Service Provider operates special routers that link networks to networks — the connective tissue of the internet.

  4. 4

    Across the backbone

    Packets hop router to router across continents and ocean-floor cables. At every hop, TCP/IP adds and reads address layers so nothing gets lost.

  5. 5

    The web server

    A server with its own unique IP address receives the request, assembles the page you asked for, and fires it back along the same route.

// protocol

TCP/IP

The internet's postal system. IP addresses every packet so routers know where it's going; TCP guarantees every packet arrives intact and in order — re-requesting any that go missing.

device ──▶ router ──▶ isp ──▶ server

04 — The Toolkit

Six tools, one workshop

You don't need much to build for the web — but you need these.

⌨️

VS Code

A text editor built for code — syntax highlighting, extensions, and an integrated terminal. Your home base.

🖥️

Command Line

Talk to your computer directly. Navigate, create, and run anything with a few keystrokes — no clicking required.

🌿

Git

Version control: a time machine for your code. Experiment freely, roll back fearlessly.

🐙

GitHub

Where your Git history lives online — portfolio, backup, and collaboration hub in one.

🌐

Chrome DevTools

Inspect, debug, and profile any page live in the browser. See exactly what your code is doing.

🧠

Stack Overflow

Someone has hit your bug before — and answered it. Learning to search well is a developer superpower.
